# CGPA Calculator - C++ Project

This is a CGPA Calculator developed in C++ as part of my CodeAlpha Internship.  
It allows users to input their subject grades and credit hours, then calculates and displays the CGPA accurately.  

## Features
- Calculates GPA and CGPA based on user input
- Handles multiple subjects and credit hours
- Simple and beginner-friendly console application

## Tech Stack
- Language: C++
- Concepts Used: Loops, Conditionals, Functions, Arrays, Input/Output
